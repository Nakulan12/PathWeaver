# 🚀 PathWeaver - Adaptive AI Onboarding Engine

> **From Resume → To Role-Ready**
> An intelligent system that transforms static onboarding into **dynamic, personalized learning pathways**.

---

## ⚡ What Makes This Special?

* **Hybrid AI Engine** (NLP + ML + Graph Logic)
* **Personalized Learning Paths** (not generic courses)
* **Readiness Scoring** (instant evaluation)
* **Explainable Decisions** (no black-box AI)

---

## 🧠 Problem

Traditional onboarding is broken:

* Same training for everyone
* Wasted time for experienced hires
* Overwhelming for beginners

➡️ Result: **Slow productivity + inefficient learning**

---

## 💡 Our Solution

PathWeaver builds a **custom learning roadmap** for every individual:

* Extracts real skills from resume
* Compares with job requirements
* Identifies exact skill gaps
* Generates **optimized learning order**

---

## ⚙️ How It Works (Simple Flow)

```
Resume + JD
      ↓
Skill Extraction (NLP + Embeddings)
      ↓
Gap Analysis
      ↓
Dependency Graph Engine
      ↓
Personalized Learning Path + Readiness Score
```

---

## 🧩 System Architecture

```
User Input (Resume + JD)
        ↓
Frontend (React UI)
        ↓
FastAPI Backend
        ↓
---------------------------------
| Skill Extraction Layer        |
| - NLP (keyword matching)     |
| - ML (embeddings)            |
---------------------------------
        ↓
Skill Gap Analysis
        ↓
---------------------------------
| Adaptive Engine              |
| - Dependency Graph           |
| - Learning Path Generator    |
---------------------------------
        ↓
Output:
- Readiness Score
- Learning Path
- Reasoning Trace
```
