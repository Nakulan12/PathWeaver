# 🎨 PathWeaver Frontend

This module provides the user interface for PathWeaver.

---

## 🖥️ Features

* Upload Resume + Job Description
* Display Readiness Score
* Show Matched & Missing Skills
* Visual Learning Path
* What-if Simulation

---

## ⚙️ Run

```bash
cd frontend
npm install
npm start
```

---

## 📌 Notes

* Built using React (JavaScript only)
* Communicates with FastAPI backend via REST API
